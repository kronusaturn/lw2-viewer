About GreaterWrong

GreaterWrong is an alternative way to browse LessWrong 2.0 (show), the Effective Altruism forum (show), and Arbital (show) with a focus on speed and usability.

Source code is available on GitHub, and issues are tracked on OborWiki.

Themes

Nine different themes are available. Click on the theme selection buttons at the top left of the page (labeled with the letters ‘A’ through ‘I’) to select a theme. (You can also change the theme in the Customize Appearance window (see below).)

Adjustable width

You can adjust the width of the content column by clicking on one of the three width selection buttons at the top-right of the page.

The options are:

  • fixed-width, narrow (the default setting) ( )
  • fixed-width, wide ( )
  • fluid width (which expands to fill the width of the browser window) ( )

Adjustable text size

You can adjust the size of the body text by clicking the text size adjustment buttons at the top-right of the page. (This changes the text size of both posts and comments.) Click “−”/“+” to decrease/increase the text size; click “A” to reset to the default text size.

(The text size adjustment buttons are hidden when viewing pages that don’t contain any body text. You can always adjust the text size setting via the Customize Appearance window (see below), regardless of what page you’re viewing.)

This feature does not currently work in Firefox (or other Mozilla-based browsers).

Appearance customization

In addition to picking a theme, you can further customize the way that GreaterWrong looks. Click on the “sliders” (  ) icon, at the top left of the page, to bring up the Customize Appearance window. The following settings are available:

  • Text size
  • Invert colors (a.k.a. photo-negative)
  • Saturation
  • Brightness
  • Contrast
  • Hue rotation (a.k.a. color-shift)

Keyboard-based navigation

GreaterWrong has extensive support for keyboard-based navigation.

Post listing quick-nav

When viewing any post listing page (e.g., the front page), you can:

  • Select the next post with the . (period) key;
  • Select the previous post with the , (comma) key;
  • Use the Enter key to go to the selected post

(These are not accesskeys; you need not hold any modifier keys to use them.)

New comments quick-nav

When viewing any post page that contains new comments (as shown by the new comment indicator on the bottom right of the page), you can:

RSS feeds

The following RSS feeds are available:

  • Home (“Frontpage posts”)
  • Featured (“Curated posts”)
  • All (“Community posts”)
  • Meta posts
  • Recent comments
  • Feeds for just the comments, just the posts, or both comments and posts, of an individual user (for example, here’s Eliezer Yudkowsky’s comments feed); these are accessible from the user’s page (e.g., here’s Eliezer Yudkowsky’s page)

Archive browser

GreaterWrong has an archive browser that lets you quickly find posts from any time in LessWrong’s past. Click the Archive link at the top of the page to go to the archive browser.

Auto-generated table of contents

Posts that contain multiple headings (such as this one) have automatically-generated tables of contents (with links), for easy navigation.

Markdown editor

GreaterWrong’s comment/post editor lets you write Markdown directly (much like the old Less Wrong site). However, to save you from having to type all the markup manually, the buttons on the editing toolbar insert each of the various types of available markup (e.g., to make text bold, add a hyperlink, insert an image, etc.) into your post or comment. (Hover over each button in the editing toolbar to see what it does.)

You can click the “Show Markdown hints” link to show a help popup, or click the “Markdown” link to go to the Markdown reference website.

When you submit a post or comment, GreaterWrong saves the raw text and markup you entered, so that you can easily edit it later.
